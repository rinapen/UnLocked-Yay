# りなぺん

## コマンドの使い方

### インストール

以下のコマンドを使用して、必要な依存関係をインストールしてください:

```bash
npm install
```

## 使用方法

### コマンド一覧

| コマンド | 説明 |
|:--------|:-----|
| **`npm run build`** | プロジェクトをビルドします |
| **`npm start`** | アプリケーションを起動します |

### 実行例

#### ビルドの実行

```bash
npm run build
```

#### アプリケーションの起動

```bash
npm start
```

## 注意事項

- コマンドを実行する前に、必ずプロジェクトのルートディレクトリにいることを確認してください
- Node.jsのバージョンは **`16.x`以上** を推奨します
